HELLO WORLD!

Nov 5, 2016

HELLO WORLD! is a web application for pair programming and video meeting. Pair programming is often used in education for programming, however, it is difficult to find a good teacher in your neighborhood. HELLO WORLD! helps them to do pair programming online.

Features:

  • Edit codes at the same time
  • Chat in video communication
  • Run codes in a browser

This project is joint work with my friends in a hackathon JPHacks 2016. This is an overview of the project and its detail is here in Japanese.
